A collection of 50 PCB design projects using KiCad, ranging from basic power supplies to advanced microcontrollers.

# 50 Projects in KiCad

## Project 1: Fixed 5V Power Supply
A simple DC-DC regulator circuit that converts 7V-12V input into a stable 5V output.

### Features:
* **L7805** Linear Regulator
* **GND Plane** for heat dissipation and signal stability
* **LED Indicator** for power status
